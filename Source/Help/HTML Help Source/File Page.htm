Log File

Depending on the target type, the log file can be found in the following locations:
- Windows: "<System Definition Folder Path>\Logs\telemetry.tdms"
- RT: "c:\ni-rt\NIVeriStand\Logs\telemetry.tdms"

By default, Overwrite File If Exists? is enabled. This will overwrite the log file during each deployment of the system definition.

Note: If Overwrite File If Exists? is not enabled, the custom device will only create a new log file if no log file exists at the specific location on the target.
