Log File

This section contains details regarding the location of the log file and lets the user choose to overwrite the log file if it already exists on the target.

The log file will be located at one of the following paths depending on the target type:
Log File Path for Windows Targets: "<System Definition Folder Path>\Logs\telemetry.tdms"
Log File Path for RT Targets: "c:\ni-rt\NIVeriStand\Logs\telemetry.tdms"

Overwriting the log file if it already exists can be done by having checked the Boolean Control checkbox:
Overwrite File If Exists?: If the checkbox is checked, the Telemetry Custom Device will overwrite the log file during each deployment of the VeriStand system. If the checkbox is unchecked the Telemetry Custom Device will create a new log file only if no log file exists at the specific location on the target. If the log file already exists, and the checkbox is unchecked, no new file will be created during the deployment.
