Telemetry Custom Device

The Telemetry Custom Device supports VeriStand benchmarking by logging system channels, and by monitoring target resources.

You can find a Quick Start Guide for this custom device at the following location:

"C:\Users\Public\Documents\National Instruments\NI VeriStand (Year)\Custom Devices\National Instruments\Telemetry\Windows\Quick Start Documentation\Telemetry Quick Start.md"

Logged Data

For each PCL iteration the custom device logs the following VeriStand System Channels:
- Absolute Time
- Iteration
- Actual Loop Rate
- HP Loop Duration
- HP Count
- LP Count
- LP Data Count
- Failure Count
- HS TCP Overflow Count
- Model Count
- WPL Error Count
- WPL Overflow Count
- WPL TCP Overflow Count

The custom device also logs data representing the Target Resources for the target that is running the VeriStand Engine. The log file will contain the following metrics:
- Physical Memory Usage (%)
- CPU x Usage (%) - where x represents the number of each CPU present on the target system.

The data will be logged into a single TDMS file created at the location described in the Log File page.
