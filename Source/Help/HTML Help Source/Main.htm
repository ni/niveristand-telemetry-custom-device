Telemetry Custom Device

The Telemetry Custom Device provides support for benchmarking a VeriStand System by logging System Channels data for each PCL iteration.

A Quick Start Guide can be found at the following location:

"C:\Users\Public\Documents\National Instruments\NI VeriStand (Year)\Custom Devices\National Instruments\Telemetry\Windows\Quick Start Documentation\Telemetry Quick Start.md"

Logged System Channels

The Custom Device will create a log file containing the following system channels:

- Absolute Time
- Iteration
- Actual Loop Rate
- HP Loop Duration
- HP Count
- LP Count
- LP Data Count
- Failure Count
- HS TCP Overflow Count
- Model Count
- WPL Error Count
- WPL Overflow Count
- WPL TCP Overflow Count

The data will be logged in the form of a TDMS file at the path described in the Log File page.
