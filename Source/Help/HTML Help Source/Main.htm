Telemetry Custom Device

The Telemetry Custom Device supports VeriStand benchmarking by helping you log system channels and monitor target resources. You can find a Quick Start Guide for this custom device at the following location:

"%Public%\Documents\National Instruments\NI VeriStand (Year)\Custom Devices\National Instruments\Telemetry\Windows\Quick Start Documentation\Telemetry Quick Start.md"

Logged Data

Data is logged into a TDMS file created at the location specified on the Log File page.

For each PCL iteration, the custom device logs the following VeriStand System Channels:

  • Absolute Time
  • Iteration
  • Actual Loop Rate
  • HP Loop Duration
  • HP Count
  • LP Count
  • LP Data Count
  • Failure Count
  • HS TCP Overflow Count
  • Model Count
  • WPL Error Count
  • WPL Overflow Count
  • WPL TCP Overflow Count

The custom device also logs Target Resources data for the target that is running the VeriStand Engine. The log file will contain the following metrics:

  • Physical Memory Usage (%)
  • CPU <XXX> Usage (%), where <XXX> represents the number of the CPU on the target system.
